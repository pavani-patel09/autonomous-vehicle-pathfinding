# Autonomous Vehicle Pathfinding Using Trees

This project simulates **autonomous vehicle navigation** using tree-based search algorithms to find optimal routes. It demonstrates **AI pathfinding logic**, decision-making, and optional visualization for autonomous vehicles in a controlled environment.

## Features
- Tree-based pathfinding for autonomous vehicles
- Optimal route computation using depth-first search (DFS)
- Optional visualization of paths
- Logging of steps and decision-making

## Technologies Used
- Python
- NumPy
- Matplotlib (for visualization)
- Scikit-learn (optional for AI heuristics)

## Project Structure
- `main.py` – Main program to run the simulation  
- `tree_utils.py` – Functions for generating tree structures and finding paths  
- `requirements.txt` – List of Python libraries needed  
- `README.md` – Project description and instructions
